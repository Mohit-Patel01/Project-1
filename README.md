# How to use <br/>
1. download the folder then open it in Your code editor open the code editor's terminal then run the command <br/>
2. npm run build <br/>
3. a folder with the name dist will be created. <br/>
4. open chrome then extenstin settings turn on developer mode its on the upper right corner <br/>
5. press load unpacked and choose the dist folder.

# What it is for

What this extension does is it can answer you question and extract text from handwritten or digital pdfs then you can save those question with answer and extracted text to your notes folder locally with just a click of button
